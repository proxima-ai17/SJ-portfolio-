# John Portfolio Website

## Files
- `index.html` → main website structure
- `style.css` → all styling
- `script.js` → interactions and effects
- `images/` → project images

## How to run
1. Open the folder in VS Code
2. Open `index.html`
3. Use Live Server if you have it installed, or just open in browser

## Replace these placeholders
Search and replace these in `index.html`:

- `YOUR_RESUME_PDF_LINK_HERE`
- `YOUR_LINKEDIN_LINK_HERE`
- `YOUR_GITHUB_PROFILE_LINK_HERE`
- `YOUR_GOOGLE_DRIVE_FOLDER_LINK_HERE`
- `YOUR_LOOM_OR_VIDEO_LINK_HERE`
- `YOUR_DOCUMENTATION_LINK_HERE`

## GitHub Pages deployment
1. Create a GitHub repository
2. Upload all files to the root of the repo
3. Go to repository Settings
4. Open Pages
5. Set source to `Deploy from a branch`
6. Select `main` branch and `/root`
7. Save
8. GitHub will generate your portfolio URL
